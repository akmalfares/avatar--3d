# الكنيسة القبطية الأرثوذكسية - موقع تفاعلي متكامل

## 🏛️ نظرة عامة

موقع تفاعلي شامل للكنيسة القبطية الأرثوذكسية يجمع بين التقنيات الحديثة والمحتوى الروحي الأصيل. يوفر الموقع تجربة تفاعلية فريدة تشمل الزيارات الافتراضية، المكتبة الرقمية، نظام الأسئلة والأجوبة، وميزة تحويل الصور إلى Avatar ثلاثي الأبعاد.

## ✨ الميزات الرئيسية

### 🏛️ الزيارات الافتراضية
- **كنيسة السيدة العذراء المعلقة**: زيارة تفاعلية بتقنية 360° من موقع 360cities.net
- **دير الأنبا أنطونيوس**: جولة شاملة في أقدم دير في العالم
- **كنيسة أبو سرجة**: استكشاف مكان إقامة العائلة المقدسة
- تقنية التنقل التفاعلي مع التعليق الصوتي

### 📚 المكتبة الرقمية
- مجموعة شاملة من الكتب والمراجع الأرثوذكسية
- تصنيف حسب الموضوعات: التاريخ، اللاهوت، سير القديسين، الصلوات، الروحانيات
- نظام بحث متقدم وفلترة
- إمكانية حفظ المفضلة وتتبع تقدم القراءة
- واجهة عرض متعددة (شبكة/قائمة)

### ❓ نظام الأسئلة والأجوبة
- قاعدة بيانات شاملة للأسئلة الشائعة
- إجابات من خبراء الكنيسة المعتمدين
- إمكانية طرح أسئلة جديدة
- تصنيف حسب الموضوعات
- نظام التقييم والمفضلة

### 🤖 إنشاء Avatar
- تحويل الصور الشخصية إلى شخصيات ثلاثية الأبعاد
- دعم رفع الصور أو التقاط بالكاميرا
- تخصيص المظهر والملابس
- تصدير عالي الجودة

### 💬 نظام المحادثات
- غرف محادثة متعددة حسب الموضوع
- محادثات فورية مع المجتمع
- نظام الرسائل الخاصة
- مشاركة الملفات والصور

### 🎮 الألعاب التفاعلية
- ألعاب تعليمية مقسمة حسب العصور التاريخية
- العصر الرسولي، عصر الاضطهاد، العصر الذهبي، العصر الإسلامي، العصر الحديث
- أنواع متنوعة: مغامرات، قصص، ألغاز، اختبارات
- تتبع التقدم والإنجازات

## 🛠️ التقنيات المستخدمة

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: CSS Grid, Flexbox, CSS Animations
- **Icons**: Font Awesome 6.0
- **Fonts**: Google Fonts (Cairo, Amiri)
- **APIs**: Camera API, File API, Local Storage
- **Responsive Design**: Mobile-first approach
- **Virtual Tours**: 360cities.net integration

## 🚀 التشغيل والتطوير

### متطلبات النظام
- Node.js 14+ 
- npm أو yarn
- متصفح حديث يدعم ES6+

### التثبيت
```bash
# استنساخ المستودع
git clone https://github.com/akmalfares/coptic-church-website.git

# الانتقال للمجلد
cd coptic-church-website

# تثبيت التبعيات
npm install

# تشغيل الخادم
npm start
```

### الوصول للموقع
افتح المتصفح وانتقل إلى: `http://localhost:3000`

## 🔧 التخصيص

### إضافة محتوى جديد
1. **كتب جديدة**: تحديث `digital-library.js`
2. **أسئلة وأجوبة**: تحديث `qa-system.js`
3. **زيارات افتراضية**: تحديث `enhanced-app.js`
4. **ألعاب**: إضافة في قسم الألعاب

### تخصيص التصميم
- تحديث متغيرات CSS في `style.css`
- تعديل الألوان والخطوط
- إضافة رسوم متحركة جديدة
